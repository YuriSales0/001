# HostMasters — Brand Guidelines

## Cores

| Token | Hex | Uso |
|-------|-----|-----|
| Deep Navy | `#0B1E3A` | Backgrounds, headers, texto principal |
| Antique Gold | `#B08A3E` | CTAs, badges, destaques, links activos |
| Ivory | `#FAF8F4` | Fundo claro de cards e secções |
| Sand | `#F2EDE4` | Fundo secundário, hover states |
| Slate | `#4A5568` | Texto secundário |
| Border | `#E2DDD4` | Bordas de cards |
| White | `#FFFFFF` | Fundo de inputs, modais |
| Success | `#1B7A4A` | Estados positivos |
| Error | `#B91C1C` | Estados de erro |

## Tipografia

| Família | Uso | Peso |
|---------|-----|------|
| **Inter Tight** | UI, labels, body, botões, nav | 400, 500, 600, 700 |
| **Cormorant Garamond** | Headings (h1, h2), preços, destaque | 500, 600, 700 |

## Regras de aplicação

- h1/h2: Cormorant Garamond (serif)
- Tudo o resto: Inter Tight (sans-serif)
- Botão primário: bg Antique Gold, texto Deep Navy, font-weight 700
- Botão secundário: border Deep Navy, texto Deep Navy
- Cards: bg Ivory, border Border, rounded-xl
- Header/sidebar: bg Deep Navy
- Focus ring: ring-2 ring-[#B08A3E]/40
